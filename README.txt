# Mr. Compilex Electronic Store

## Requirements
- XAMPP/WAMP/MAMP (Apache + MySQL + PHP)

## Installation Instructions

1.  **Database Setup:**
    - Open PHPMyAdmin (http://localhost/phpmyadmin).
    - Create a new database named `mr_compilex_store`.
    - Import the file `database/schema.sql` into this database.

2.  **Project Setup:**
    - Copy the `scratch` folder (or the project root) to your XAMPP `htdocs` directory.
    - Rename the folder to `electronic-store` (optional).

3.  **Run the Project:**
    - Open your browser and go to `http://localhost/electronic-store/`.

## Login Credentials
